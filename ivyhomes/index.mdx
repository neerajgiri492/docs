# IvyHomes Platform Documentation

Welcome to the IvyHomes platform documentation. This guide covers the technical implementation, architecture, and development workflows for our real estate platform.

## Platform Overview

IvyHomes is a comprehensive real estate platform built with:
- **Frontend**: Next.js 15 with React 19 and TypeScript
- **Backend**: Django REST Framework with PostgreSQL
- **UI Framework**: Chakra UI v3 with custom theming
- **Authentication**: JWT-based with persistent state management
